  • All random numbers must be rounded to 3 decimal places
  • Decimal sum must be rounded to 3 decimal places
  • Fractions must have denominators ≤ 1000
  • Program must use random.seed(int(input("Enter the seed: ")))
  • File should be named py1_team_2_teamnumber.py

Random Number Ranges

  • First number: 0 to 100
  • Second number: 10 to 50
  • Third number: 20 to 40
  • Fourth number: 100 to 200

Expected Output Format

  • Display each random number with 3 decimal places
  • Show decimal sum calculation: "Sum from decimals: X + Y + Z + W = Total"
  • Show fraction sum calculation: "Sum from fractions: A/B + C/D + E/F + G/H = Result"
